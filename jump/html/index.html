dSFMT-jump Documentation

This is SIMD oriented Fast Mersenne Twister pseudorandom number generator(dSFMT) with jump function. This Project provides jump function of dSFMT.

There are two steps for jumpping.

  1. Calculate jump polynomial using magic polynomial.
    • calc-jump executable file calculates a jump polynomial.
    • calc-jump is written in C++ language and needs NTL for compile.
    • Additionally, users can call dsfmt::calc_jump(std::string& jump_str, NTL::ZZ& step, NTL::GF2X& characteristic) in SFMT-calc-jump.hpp from their programs.
    • jump steps used above is based on internal state, i.e, doubles are skipped 2 * jump steps.
  2. Change internal state of dSFMT into jumpped state.
    • dSFMT_jump(dsfmt_t * dsfmt, const char * jump_str) changes internal state of dsfmt in user's programs.
    • See also:
      sample1.c sample2.c

Caution: The jump polynomial calculated using a magic polynomial is only available for jumpping dSFMT states which has MEXP correspond to the magic polynomial.
